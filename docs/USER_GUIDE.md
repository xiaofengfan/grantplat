## 2. 功能模块

### 2.1 工作台 (Dashboard)

工作台是系统的首页，展示最重要的信息。

**主要功能：**
- 查看总资产和今日收益
- 监控自选股票池行情
- 查看交易提醒
- 查看AI交易信号
- 快速跳转各功能模块

### 2.2 股票池

管理您的自选股票。

**添加股票：**
1. 点击右上角「添加股票」按钮
2. 输入股票代码（如：600519）
3. 选择市场和分组
4. 点击确认

**设置价格提醒：**
1. 点击股票行的「闹钟」图标
2. 选择提醒类型（价格/涨跌幅/成交量）
3. 设置触发条件（高于/低于）
4. 输入阈值数值

**AI分析：**
1. 点击股票行的「AI分析」按钮
2. 自动跳转到AI助手页面
3. 查看该股票的实时行情
4. 进行技术分析/基本面分析/风险评估

### 2.3 策略管理

提供8种预设量化策略。

**策略列表：**
| 策略名称 | 类型 | 说明 |
|---------|------|------|
| 双均线策略 | 零代码 | 快速均线与慢速均线交叉产生信号 |
| MACD趋势策略 | 模板 | MACD指标背离和交叉信号 |
| RSI超买超卖 | 模板 | RSI超买超卖区域逆势交易 |
| 布林带突破 | 全代码 | 突破布林带上下轨交易 |
| 网格交易 | 零代码 | 等间距网格高抛低吸 |
| 北向资金流向 | 模板 | 跟踪沪深港通资金流向 |
| 价值投资选股 | 全代码 | 基于财务指标筛选低估股 |
| 行业轮动 | 模板 | 根据经济周期轮动切换行业 |

**使用策略：**
1. 在策略列表查看各策略表现
2. 点击策略名称查看详情
3. 点击「回测」进行历史验证
4. 点击「编辑」调整参数

### 2.4 自动交易

自动化策略执行引擎。

**启动自动交易：**
1. 进入「自动交易」页面
2. 点击「启动引擎」
3. 添加要运行的策略
4. 查看生成的交易信号
5. 点击信号后的「执行」确认下单

**支持的策略类型：**
- `ma_cross` - 均线交叉策略
- `rsi` - RSI超买超卖策略
- `bollinger` - 布林带策略
- `macd` - MACD策略

### 2.5 AI助手

AI智能股票分析助手。

**主要功能：**
- 输入股票代码进行AI分析
- 技术分析（支撑位/压力位/MACD/KDJ）
- 基本面分析（财务指标/行业地位）
- 风险评估（市场/经营/政策风险）
- 投资建议（买入/卖出/持有）

**使用方法：**
1. 输入股票代码
2. 点击「AI分析」按钮
3. 选择分析类型（技术/基本面/风险/建议）
4. 查看AI分析报告
5. 根据建议决定是否交易

**快捷功能：**
- 📊 市场趋势分析
- 🏆 白马股推荐
- 📈 技术指标解释
- 🛡️ 策略风险分析
- 📜 政策解读

### 2.6 实盘交易

类似同花顺的交易界面。

**主要功能：**
- 左侧自选股列表
- 中间行情图表（分时/K线）
- 交易面板（买入/卖出）
- 右侧持仓/委托/成交
- 钱包/账单/收益管理

**下单流程：**
1. 选择股票
2. 输入价格和数量
3. 点击「买入」或「卖出」
4. 确认下单信息
5. 查看成交结果

### 2.7 回测验证

Tick级高精度回测引擎。

**回测流程：**
1. 选择要回测的策略
2. 设置回测时间范围
3. 选择回测股票
4. 设置初始资金
5. 点击「开始回测」
6. 查看回测报告（收益率/夏普比率/最大回撤）

### 2.8 交易管理

管理您的资金和交易记录。

**钱包：**
- 查看总资产/可用现金/持仓市值
- 充值/提现功能
- 资金变动记录

**账单：**
- 查看所有交易流水
- 筛选特定股票/日期
- 统计手续费

**收益：**
- 查看每日收益
- 收益曲线图
- 资产分布图

### 2.9 风险监控

风控规则配置和监控。

**风控规则：**
- 单股最大持仓比例
- 日亏损限制
- 最大回撤限制
- 单笔最大交易量

**预警通知：**
- 风险超标实时提醒
- 持仓集中度预警
- 亏损预警

### 2.10 系统设置

系统配置和用户管理。

**用户管理：**
- 添加/编辑/删除用户
- 设置用户角色（管理员/交易员/查看者）
- 启用/禁用用户

**接口配置：**
- 行情API（新浪/腾讯/东方财富）
- AI API（DeepSeek/OpenAI）
- 交易API配置

**系统参数：**
- 策略参数（最大策略数等）
- 回测参数（最大回测天数等）
- 风控参数（日亏损限制等）

## 3. 常见问题

### Q1: 无法登录？
**A:** 检查账户密码是否正确，默认账户 admin@quantmaster.com / admin123

### Q2: 股票行情不更新？
**A:** 点击「刷新」按钮手动刷新，或检查网络连接

### Q3: AI分析失败？
**A:** 检查AI配置中的API密钥是否正确，或在设置中配置有效的DeepSeek/OpenAI密钥

### Q4: 如何添加新的量化策略？
**A:** 在「策略管理」页面点击「新建策略」，支持零代码/低代码/全代码三种方式

### Q5: 自动交易安全吗？
**A:** 默认情况下自动交易需要手动确认信号后才能执行，建议先在模拟环境测试

### Q6: 如何对接真实券商？
**A:** 当前版本为模拟交易，真实交易接口需要额外对接券商API（规划中）
